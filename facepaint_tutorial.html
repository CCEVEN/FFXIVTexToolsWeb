Combining Face Paints

In this tutorial I'll be combining eye shadow with freckles
First we open both files we're going to combine in Photoshop


We then copy and paste one of the textures on top of the other, you'll want to paste what you want to add on top of the texture you are looking to replace.
In my case I'll be pasting on top of the freckles texture, so that when I select freckles in game it will have both.

This will create 2 layers, one for each of the textures
Now, there are several ways to do this, but essentially we want to delete the black part of the top layer
I used the magic wand tool, set the tolerance to 15 and selected anywhere black


I then just hit Delete on the keyboard
Which leaves me with this


Then in the Layer menu we either Merge Visible or Flatten Image
All that's left to do now is save as usual, File > Save as, change Save as type to D3D/DDS, overwrite existing, and save as a DXT1
Then import in the application.
Here's the end result in-game
