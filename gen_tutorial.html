Tools Needed

  • Adobe Photoshop
  • DDS Plugins

    • Nvidia Texture Tools Plugin: https://developer.nvidia.com/nvidia-texture-tools-adobe-photoshop
    Intel Texture Tools Plugin: https://gametechdev.github.io/Intel-Texture-Works-Plugin/

    Note: Both are needed in order to load and save color files properly

    The Application

    When opening the application for the first time you will be asked to locate your FFXIV data folder.
    This is where the game installs by default:
    C:\Program Files (x86)\SquareEnix\FINAL FANTASY XIV - A Realm Reborn\game\sqpack\ffxiv


    Explorer

    After a very short loading time the list of textures will populate.
    You will find that the list is sorted into categories including: equipment slots, character, summons, minions, etc.
    The items within these categories are sorted alphabetically to make it easier to find certain textures.
    Note: The Extra category seems to be food items which may have actual in game models, but I haven't looked into it as of yet so you can only view the names.

    Search

    You may also use the search function accessed from the Search > Find menu for quick access to specific textures.


    Go To will take you to the texture selected.

    After Selecting an Item

    When a texture is selected and loaded, the default Race, Texture Map, and Part will be selected.
    If there is only 1 available item to select, the drop down will be disabled. This makes it easier to determine if there are multiple Races, Texture Maps, or Parts for that item.


    Race
  • The application will determine if the item has multiple textures associated with specific races and add those races to the drop down
  • It is important to note, that most textures are shared among all races, in which case the default race will be selected and the drop down menu will be disabled.
  • Midlander Male is the common default race for Male Only and Regular equipment textures
  • Midlander Female is the common default race for Female Only equipment textures

    • Texture Map
    There are 4 types of available textures and a ColorSet that can appear in the Texture Map drop down.

  • Diffuse - A type of color map, in FFXIV a stain or dye can sometimes go over or get mixed with this texture
  • Normal - Gives the illusion of height based on light reflections
  • Specular - Affects the reflectiveness or how shiny a texture is
  • Mask - A layer type texture that allows changes without modifying the original texture
  • ColorSet - A 4x16 texture that determines the colors of a model, the location of the color in the model is determined by the alpha channel of the Normal Map

    • For more information on Texture Maps this is a good place to start:
    http://blog.digitaltutors.com/understanding-difference-texture-maps/

    Part/Num
  • This dropdown shows the part or number for some models have different parts to them, such as Head+Body category, crafting tools, character faces, and more.

    • Saving, Importing, and Reverting

    The options available for each texture:


  • Save PNG - Saves the texture as a regular .png file, used if you want to upload to a site and show the texture
  • Save DDS - Saves the texture as a DirectDraw Surface, this is the file we modify in Photoshop.
  • Import - Once a texture has been modified, clicking import will write it into the new .dat3 file and update the index file to point to it
  • Revert - This will update the index file to point at the original texture again

    • Note: The revert option will remain active even if the game is using the original texture. To check if a texture is modified, see the Active column in the Mod List window.
    Once you've selected a texture you would like to modify you may can click on save DDS to save the texture file.

    The saved DDS file will be located within the FFXIV TexTools folder in this format:
    Example: FFXIV TexTools/Saved/Body/Hempen Tabard/v15_c0101e0011_top_n.dds
    Note: Save PNG will save to the same location.

    The Info box

    This will show you relevant information about the texture being viewed.

    The most important thing to note is the Texture Type, this will tell you in what format you must save the texture from Photoshop.

    Settings



  • Directories - Modify your ffxiv and save location
  • Revert All - Modify index file to point to the original textures for anything that has been modified. Recommend using before any major patch
  • Reapply All - Modify the index file to point to all the modified textures again.
