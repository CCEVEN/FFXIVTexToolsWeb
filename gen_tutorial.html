Tools Needed

  • Adobe Photoshop
  • DDS Plugins

    • Nvidia Texture Tools Plugin: https://developer.nvidia.com/nvidia-texture-tools-adobe-photoshop
    Intel Texture Tools Plugin: https://gametechdev.github.io/Intel-Texture-Works-Plugin/

    Note: Both are needed in order to load and save color files properly
